A simple and light-weight plugin for sending email notification on reply to a comment.It uses the basic wp_mail function.

== Description ==
This plugin adds a checkbox to the comment form, if someone comments and checks the checkbox then they receive the response to their comments after it is approved.
In admin panel there is a setting section with options to enable/disable the checkbox by default,change the label for the checkbox and customize the email with tags to include commenter_name, post_title, post_link, sender_name, reply_content . 

## Requirements
* Wordpress 2.9.2 or higher
* PHP 5.1 or higher
* MySQL 5.x or higher
* Threaded/Nested comments option Enabled

## Main Features
*very Simple to use
*Bug free
*compatible with other similar plugins


== Installation ==

1. Upload the entire folder and all the subfolders to your Wordpress plugins folder
2. Activate it
